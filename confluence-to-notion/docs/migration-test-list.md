# Confluence DC → Notion ページ移行テストリスト

## 凡例

### Notion 対応

| 記号 | 意味 |
|------|------|
| ○ | 完全対応（ほぼ同等の機能がある） |
| △ | 部分対応・代替あり（代替手段があるが完全ではない） |
| ✕ | 対応困難（Notion では実現が困難、または大幅な変換が必要） |

### 移行難易度

| 記号 | 意味 |
|------|------|
| L | Low — 自動変換可能、または単純なマッピング |
| M | Medium — 変換ロジックが必要、または手動確認が必要 |
| H | High — 大幅な再設計が必要、または移行不可能 |

### テスト結果の記入方法

テスト後に「テスト結果」列を以下で更新する。

| 記号 | 意味 |
|------|------|
| PASS | 期待どおりに移行・再現できた |
| PARTIAL | 一部再現できた（備考に詳細を記入） |
| FAIL | 移行・再現できなかった |
| SKIP | 対象環境で未使用のためスキップ |
